# Personal Website Starter

This is a simple Jekyll website designed for GitHub Pages and inspired by academic personal pages such as AcademicPages.

## Structure

- `index.md`: home page
- `publications.md`: publications list
- `teaching.md`: teaching page
- `cv.md`: CV page
- `_layouts/default.html`: shared layout
- `assets/css/main.css`: site styling

## Customize

1. Update `_config.yml` with your name, email, and site details.
2. Replace placeholder text in the Markdown pages.
3. Add your profile image to `assets/images/profile.jpg`.
4. Add your CV PDF to `files/cv.pdf`.

## Deploy To GitHub Pages

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload these files to that repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/ (root)` folder.

GitHub Pages will build the site automatically.
